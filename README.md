# ⚡ Асинхронность в JavaScript — практические задания

Проект демонстрирует работу **Event Loop**, **Promise**, **async/await** и визуализацию асинхронных процессов через прогресс-бар.

---

## Структура проекта

```
async-js-hw/
├─ task1_event_loop.js    # Задание 1 — Event Loop
├─ task2_promises.js      # Задание 2 — Цепочка Promise
├─ task3_async_await.js   # Задание 3 — async/await
└─ task4_progress.html    # Задание 4 — Прогресс-бар
```

---

## Задание 1. Понимание Event Loop

Скрипт выводит порядок выполнения синхронного кода, микротасков и макротасков.
```bash
node task1_event_loop.js
```

---

## Задание 2. Работа с Promise

Эмуляция асинхронных запросов с задержкой и обработкой ошибок через `.catch()`.
```bash
node task2_promises.js
```

---

## Задание 3. Работа с async/await

Переписанная версия задачи 2 с использованием `async/await` и функции `delay(ms)`.
```bash
node task3_async_await.js
```

---

## Задание 4. Прогресс-бар

HTML-страница, визуализирующая выполнение задачи (5 секунд, обновление каждую секунду).
Нужно открыть `task4_progress.html` в браузере.
